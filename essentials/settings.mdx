Every merchant needs a set of global settings to apply the correct taxes. This is where you can set the default tax rates, tax rules, and tax classes. You can also set the default tax settings for your merchant.

## Properties

<ResponseField name="OSS" type="boolean" required>
Whether you are using the One Stop Shop scheme for VAT. Read more on the european commission's [website](https://ec.europa.eu/taxation_customs/business/vat/telecommunications-broadcasting-electronic-services/one-stop-shop_en).

</ResponseField>
