## Setup your development

Learn how to quickly integrate our API to start applying correct taxes and get a complete overview of your sales taxes

### Create an API Key

<AccordionGroup>
  <Accordion icon="key" title="Create an API Key">
    In your [settings](https://app.vatr.co/settings) you can create an API key.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Bearer Authentication">
    The [Vatr.co API](api-reference) is authenticated using a Bearer authentication header ``` --header 'Authorization: Bearer {Key}' \ ```
    
  </Accordion>

</AccordionGroup>

### Start Requesting

```bash curl

curl --request POST \
     --url https://api.vatr.co/Customers \
     --header 'Authorization: Bearer {apiKey}' \
     --header 'Accept: application/json' \
     --header 'Content-type: application/json' \
     --data '
{
  "FullName": "Name",
  "VatNumber": "DK12345678",
  "CompanyName": "The company name"
}
'

```
